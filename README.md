# 情绪便利贴 - 情侣情绪管理应用

一款专为情侣设计的情绪管理应用，帮助情侣更好地理解和管理彼此的情绪。

## 功能特点

### 🎮 四关游戏系统
- **第一关：相遇** - 记录初次相遇的美好
- **第二关：初见** - 珍藏难忘的瞬间
- **第三关：期许** - 规划美好的未来
- **第四关：相爱总会有阴天** - 学习和好的艺术

### 📝 情绪便利贴
- 记录日常情绪
- AI智能回复（温柔体贴）
- 双方共享查看
- 留言互动功能
- 语音便利贴

### 🐾 虚拟宠物系统
- 每10次消气升1级
- 100级满级后重生
- 可无限重生
- 记录重生次数

### 👥 情侣功能
- 伴侣绑定
- 查看回忆（对方游戏数据）
- 个人资料管理
- 在一起天数统计
- 解绑功能（24小时冷静期）

### 📊 情绪统计
- 情绪分布分析
- 消气率统计
- 宠物成长记录

## 技术栈

### 前端
- Flutter
- Provider (状态管理)
- Dio (网络请求)

### 后端
- Python FastAPI
- MongoDB
- DeepSeek AI (智能回复)

## 快速开始

### 前端运行
```bash
cd emotion_notes_app
flutter pub get
flutter run
```

### 后端运行
```bash
cd backend
pip install -r requirements.txt
python -m uvicorn main:app --reload
```
